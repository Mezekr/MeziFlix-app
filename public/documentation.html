MeziFlix movie-api Documentation.

This web application will enable users to access information about different films, directors and genres. Users will be able to register, update their personal information and create a list of their favourite films.

Movie-API endpoints

Request URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single Movie by name or title /movies/[title] GET None A JSON object that holds data about a single film, including an ID, a name or title, a description, a genre, a director, and an image URL property.
Get the genre (description) of a movie by its name or title /movies/genres/[genrename] GET None A text about the genre's description of the movie
Get data about a director (biography, year of birth, year of death) by the name of a movie /movies/directors/[directorname] GET None A JSON object containing data about the movie's director, including biography, birthyear, and deathyear properties.
Add/register a new user /users POST A JSON object containing data about the new user being registered, structured like
{ "username" :"username", "userEmail":"username@email.com", "favouritesMovies": [] } 		A text message indicating that the new user is registered or added.
User update his/her user data /users/[ID]/[newUsername] PUT None A text message indicating the username was updated.
User add a movie to his/her list of favourites. /users/[favourites]/[ID]/[title] PUT None Text message notifying user that movie has been added to user's favourites.
User removes a movie from his/her list of favourites. /users/[favourites]/[ID]/[Title] DELETE None Text message notifying user that movie has been removed from user's favourites.
User Unregister /users/[ID] DELETE None Text message notification that a user email has been removed.

Add movie to favourites

Users add a movie to their favourite movies

URL: /users/favorite

Method: PUT

Header: None

Parameters: id (user id, Required), title (movie title to be added, Required)

Body: None

Return type: JSON

Success Response

code: 200 OK

Body Response

                    {
                        "id": 2,
                        "username": "Daniel",
                        "favouriteMovies": [
                            "Avatar"
                        ]
                    }

Delete movie to favourites

Users removes a movie from their favourite movies

URL: /users/favorite

Method: DELETE

Header: None

Parameters: id (user id, Required), title (movie title to be removed, Required)

Body: None

Return type: String

Success Response

code: 200 OK

Body Response

Avatar removed from your Favourite Movies list
